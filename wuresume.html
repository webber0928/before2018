自傳

您好！我叫吳承翰，生長在一個幸福美滿的家庭，有一個妹妹，從小爸媽對我們的管教就很嚴厲，教導我們要養成積極踏實心態，因此也讓我面對任何事物都抱持著積極跟踏實的態度。

長榮大學資訊管理學系畢業，專案主題為「銀光任務─珍藏燕巢」，因曾於高雄燕巢擔任資訊志工，有感於燕巢地區受八八水災使得當地文物遭到破壞，因此想利用數位典藏的方式保存該地區文物，使用的技術是來自於微軟一套名為「silverlight」，是一套Web應用程式開發框架，也意外開啟對web的興趣。

第一家任職於晟鑫科技公司，是一間完全使用 open sourse 的公司，讓我對linux開始產生了興趣，寫過一些shell script跟makefile。由於該公司有與政府合作開發電子書包的專案，因此也寫過一些電子書，例如 node-webkit 包靜態網頁開發可閱讀之數位課本，或是用 python 的 framework 包成 epub，讓我在學習HTML5, CSS3, jQuery的應用更加熟悉，以及搭配Bootstrap的 framework 讓頁面設計更加快速，也會使用 python 的 flask 來寫一個 web-server，並搭配mongoDB 使用。

第二家任職於眾能數位有限股份公司，負責web前端的部分，包括php, aspx切板，php的套版( joomla2.5, opencart, wordPress)，並且運用很多Html Dom的JQuery操作，來達成客戶需求上視覺絢麗的效果 以及一些防呆機制。

第三家任職於( NOWnews )今日傳媒股份有限公司，以Nodejs做主要撰寫語言，負責新聞前台的部分，以讀取MongoDB的資料讓畫面做呈現，處理廣告商上刊廣告的問題，新聞的api的撰寫維護以及網站的維護，亦在此一工作時間也熟悉了一些 Nodejs 的開發。

對於我的未來目標，我的人生規劃希望能追求更大的發展空間，不想停滯不前，而在接觸過前後端，也更加深後端程式對我的重要性，我希望早日貢獻自身力量成為公司前進的一股助力，也將是我個人職場生涯重要里程的新起點，謝謝。
