Hello Hexo

這是我的第一個blog, 對於為什麼選擇hexo做我的blog呢？
因為這可是nodejs + markdown 所組成的blog！！
是不是很潮呀~!
廢話不多說趕快看如何啟動吧

快速啟動

首先安裝 nvm + nodejs (ubuntu) :

nvm

新增以下命令列到 .bashrc 的最後:

nodejs

接著安裝hexo:

hexo

接著只需要幾個簡單的命令, 你就可以打理好一切


真的超簡單的吧！
