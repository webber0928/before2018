HTML5 Tutorial

What is HTML?

HTML是一種優越的語言，在web瀏覽器上顯示資訊，也有其他類似的語言 (e.g : XML)但是目前最常用的還是HTML。

HTML在負責兩台電腦之間的溝通(Server, Client)之間的溝通和資訊傳遞， 總體來說Server是資訊的提供者，而Client是資訊的索求者。

What is HTML5?

HTML5和HTML4.01的Tag並不完全一樣，例如<font>這一個Tag在HTML5已經不再使用， 但是在HTML4.01仍可以使用，

HTML5 Tag

  • <!– 這是註解 –>

    在HTML中加入註解，不會顯示於頁面

  • <!DOCTYPE html>

    宣告此文件的程式碼為html，

  • <a href=’http://......’>某頁面</a>

    建立一個超連結使可以連結到另一個頁面

  • <abbr>.....</abbr>

    縮寫標籤，如：HTML

  • <address>......</address>

    網路位址或地址等資料，這個tag裡不能包含<article>, <aside>, <nav>, <section>, <header>, <footer>, <hgroup>, <h1>-<h6> 或其他的 <address> 標籤

  • <area>...</area>

    定義圖片的區塊座標

  • <span></span>

    使用於inline 的段落 grouping 或 css 控制

  • <div></div>

    區塊的grouping 或 css 控制

  • <fieldset> form element</fieldset>

    form element grouping tag

  • <legend></legend>

    <fieldset> 或是 <detail> <figure> 這三類型tag使用時，可以用<legend>標示這三個內容的標題
